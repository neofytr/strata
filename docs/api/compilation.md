# Compilation

These functions handle compiling source files to object files, parallel compilation, and linking.

---

### `neo_compile_to_object_file`

> Compile a single C source file to an object file.

```c
bool neo_compile_to_object_file(neocompiler_t compiler, const char *source,
    const char *output, const char *compiler_flags, bool force_compilation);
```

**Parameters:**
- `compiler` -- the compiler to use (`NEO_GCC`, `NEO_CLANG`, etc., or `NEO_GLOBAL_DEFAULT`).
- `source` -- path to the source file.
- `output` -- path to the output object file, or `NULL` to auto-derive from the source name (e.g., `main.c` becomes `main.o`).
- `compiler_flags` -- additional flags as a string (e.g., `"-Wall -O2"`), or `NULL` for none.
- `force_compilation` -- if `true`, compile even if the output is up to date.

**Returns:** `true` on success, `false` on failure.

**Example:**

```c
// Auto-derive output name (hello.c -> hello.o)
neo_compile_to_object_file(NEO_GCC, "hello.c", NULL, "-Wall -Wextra", false);

// Explicit output path
neo_compile_to_object_file(NEO_CLANG, "src/main.c", "build/main.o", "-O2", false);

// Force recompilation
neo_compile_to_object_file(NEO_GCC, "hello.c", NULL, NULL, true);
```

### `neo_compile_parallel`

> Compile multiple source files in parallel.

```c
int neo_compile_parallel(neocompiler_t compiler, const char **sources,
    size_t count, const char *compiler_flags, bool force_compilation);
```

**Parameters:**
- `compiler` -- the compiler to use.
- `sources` -- array of source file paths.
- `count` -- number of source files.
- `compiler_flags` -- additional flags, or `NULL`.
- `force_compilation` -- if `true`, compile all files regardless of timestamps.

**Returns:** `0` on success, non-zero if any compilation failed.

**Example:**

```c
const char *srcs[] = {"src/main.c", "src/util.c", "src/parser.c"};
int rc = neo_compile_parallel(NEO_GCC, srcs, 3, "-Wall -O2", false);
if (rc != 0) {
    NEO_LOGF(NEO_LOG_ERROR, "Compilation failed");
}
```

The number of parallel jobs is controlled by `neo_set_jobs()`.

### `neo_link`

> Link one or more object files into an executable. This is a variadic macro that automatically terminates the object list with `NULL`.

```c
#define neo_link(compiler, executable, linker_flags, forced, ...) \
    neo_link_null((compiler), (executable), (linker_flags), (forced), __VA_ARGS__, NULL)
```

**Parameters:**
- `compiler` -- the compiler/linker to use.
- `executable` -- output executable name.
- `linker_flags` -- additional linker flags (e.g., `"-lm -lpthread"`), or `NULL`.
- `forced` -- if `true`, re-link even if the output is up to date.
- `...` -- one or more object file paths.

**Returns:** `true` on success, `false` on failure.

**Example:**

```c
neo_link(NEO_GCC, "myapp", "-lm -lpthread", false, "main.o", "util.o", "parser.o");
```

---

### `neo_link_null`

> Link object files into an executable with an explicit `NULL` terminator. This is the underlying function used by `neo_link`.

```c
bool neo_link_null(neocompiler_t compiler, const char *executable,
    const char *linker_flags, bool forced_linking, ...);
```

**Parameters:**
- `compiler` -- the compiler/linker to use.
- `executable` -- output executable name.
- `linker_flags` -- additional linker flags, or `NULL`.
- `forced_linking` -- if `true`, always re-link.
- `...` -- object file paths, terminated by `NULL`.

**Returns:** `true` on success, `false` on failure.

**Example:**

```c
neo_link_null(NEO_GCC, "myapp", "-lm", false, "main.o", "util.o", NULL);
```

### `neo_needs_recompile`

> Check whether a source file needs recompilation by examining header dependencies.

```c
bool neo_needs_recompile(const char *source, const char *output);
```

**Parameters:**
- `source` -- path to the source file.
- `output` -- path to the output object file.

**Returns:** `true` if the source or any of its included headers are newer than the output (or if the output does not exist). `false` if the output is up to date.

**Example:**

```c
if (neo_needs_recompile("main.c", "build/main.o")) {
    neo_compile_to_object_file(NEO_GCC, "main.c", "build/main.o", "-O2", true);
}
```

This function parses the `.d` dependency file generated by the compiler's `-MMD` flag. If no `.d` file exists, it conservatively returns `true`.
